## 📋 文档概述

本文档详细说明 Vlinders-API 与 Vlinders-Server 之间的内部通信协议，包括认证机制、请求格式、响应格式、错误处理等。

---

## 🎯 协议定位

### 核心定位

> **内部服务间通信协议**
>
> Vlinders-API 作为协议层，负责将 Copilot API 请求转换为 Vlinders-Server 的内部请求。

### 设计原则

1. **安全第一** - 使用内部密钥认证，不暴露给外部
2. **高性能** - 支持 HTTP/2 和 gRPC，减少延迟
3. **可靠性** - 完善的错误处理和重试机制
4. **可观测** - 详细的日志和追踪
5. **向后兼容** - 协议版本化，支持平滑升级

## 🔐 认证机制

### 内部密钥认证

**机制**: 使用共享密钥进行认证

```typescript
// Vlinders-API 发送请求
const response = await fetch('http://vlinders-server:8000/internal/chat', {
  method: 'POST',
  headers: {
    'X-Internal-Auth': process.env.INTERNAL_SECRET,
    'X-Request-ID': generateRequestId(),
    'X-User-ID': userId,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(request)
});
```

```python
# Vlinders-Server 验证请求
from fastapi import Header, HTTPException

async def verify_internal_auth(
    x_internal_auth: str = Header(...),
    x_request_id: str = Header(...),
    x_user_id: str = Header(None)
):
    """验证内部认证"""
    if x_internal_auth != settings.INTERNAL_SECRET:
        raise HTTPException(
            status_code=403,
            detail="Invalid internal authentication"
        )

    return {
        "request_id": x_request_id,
        "user_id": x_user_id
    }
```

## 📡 通信协议

### HTTP/2 (主要)

**优势**:
- 多路复用，减少连接开销
- 头部压缩，减少带宽
- 服务器推送（可选）

**配置**:
```typescript
// Vlinders-API
import http2 from 'http2';

const client = http2.connect('https://vlinders-server:8000');
```

### gRPC (可选)

**优势**:
- 更高性能
- 强类型定义
- 双向流式传输

**Proto 定义**:
```protobuf
syntax = "proto3";

package vlinders;

service InferenceService {
  rpc Chat(ChatRequest) returns (stream ChatResponse);
  rpc RunAgent(AgentRequest) returns (AgentResponse);

  string model = 1;
```

### 重试策略

```typescript
// Vlinders-API 重试逻辑
async function callVlindersServer(
  endpoint: string,
  request: any,
  options: {
    maxRetries?: number;
    retryDelay?: number;
  } = {}
): Promise<any> {
  const maxRetries = options.maxRetries ?? 3;
  const retryDelay = options.retryDelay ?? 1000;

  for (let attempt = 0; attempt <= maxRetries; attempt++) {
    try {
      const response = await fetch(
        `${VLINDERS_SERVER_URL}${endpoint}`,
        {
          method: 'POST',
          headers: {
            'X-Internal-Auth': INTERNAL_SECRET,
            'X-Request-ID': generateRequestId(),
            'Content-Type': 'application/json'
          },
          body: JSON.stringify(request)
        }
      );

      if (response.ok) {
        return await response.json();
      }

      // 4xx 错误不重试
      if (response.status >= 400 && response.status < 500) {
        throw new Error(`Client error: ${response.status}`);
      }

      // 5xx 错误重试
      if (attempt < maxRetries) {
        await sleep(retryDelay * Math.pow(2, attempt));
        continue;
      }

      throw new Error(`Server error: ${response.status}`);

    } catch (error) {
      if (attempt === maxRetries) {
        throw error;
      }
      await sleep(retryDelay * Math.pow(2, attempt));
    }
  }
}
```

```typescript
const redis = new Redis();

async function getCachedInference(
  cacheKey: string,
  inferenceFunc: () => Promise<any>
): Promise<any> {
  // 检查缓存
  const cached = await redis.get(cacheKey);
  if (cached) {

  // 执行推理
  const result = await inferenceFunc();

  // 缓存结果（5 分钟）

  return result;
```
